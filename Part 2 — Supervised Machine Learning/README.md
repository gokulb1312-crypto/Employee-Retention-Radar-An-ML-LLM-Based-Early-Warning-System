# Part 2 — Supervised Machine Learning

Trains a baseline logistic-regression classifier and a cross-validated tuned random forest for employee attrition. Run Part 1 first.


Outputs include `attrition_model.joblib`, `metrics.json`, a confusion matrix/ROC plot, and feature importances. See `explain.txt` for the metric rationale.
